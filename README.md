# Colored In

Colored In helps you create better color palettes faster, with AI-powered generation, image analysis, and a workflow built for real design decisions, not random color shuffling.

## Install the Chrome Extension (Top Priority)

Use this folder only: `Chrome Extension Colored-In`

1. Open `chrome://extensions` in Chrome.
2. Turn on `Developer mode` (top right).
3. Click `Load unpacked`.
4. Select the folder `Chrome Extension Colored-In`.

Important: do not load the repo root or a different folder. Load only `Chrome Extension Colored-In`.

## Why Use Colored In

- Generate palettes with clear direction instead of guessing.
- Analyze images and links to extract or improve real-world color systems.
- Save assets and palettes in one place for faster iteration.
- Move from inspiration to production-ready color choices with less back-and-forth.

## Why Colored In Over Coolors

- Colored In focuses on guided outcomes (AI + analysis), while Coolors is mostly fast randomization.
- Colored In connects assets, palette generation, and refinement in one workflow.
- Colored In is built for creators who want intent and context, not just shuffled color combinations.
- Colored In supports extension-based color workflows directly in Chrome.

## What You Can Do

- Create palettes from prompts with AI.
- Analyze and extract palette candidates from images or websites.
- Improve existing palettes with contextual suggestions.
- Save and reuse assets and palettes across your creative process.

## Who It Is For

- Designers building brand systems.
- Content creators and social media teams.
- Founders and marketers shaping visual identity.
- Creative teams who need faster, consistent color decisions.
